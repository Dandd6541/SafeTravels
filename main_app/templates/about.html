About SafeTravels

SafeTravels is a web application that allows users to plan trips safely according to current COVID-19 data. Users have access to COVID-19 statistics from all counties in all 50 states. Users can utilize our data to keep track of upcoming and/or previous trips or destinations to monitor potential exposure.


Technologies Used

SafeTravels is a full-stack Django application that connects to and performs data operation on a PostgreSQL database. The project includes full-CRUD data operations while allowing users to authenticate and create their own unique accounts.Learn More

SafeTravels uses an API from disease.sh for COVID-19 data

This API provides a big range of detailed information about multiple viruses. From COVID-19 global data overviews to city/region specific mobility data, and data on the current outbreak of Influenza. Official government data for some countries (including US) is provided.
